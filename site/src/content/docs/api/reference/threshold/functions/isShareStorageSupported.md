# Function: isShareStorageSupported()

> **isShareStorageSupported**(): `boolean`

Returns whether the current runtime exposes the minimum capabilities required
for wrapped-share storage.

## Returns

`boolean`

`true` when Web Crypto is available and IndexedDB exists.
