# Function: hexToBytes()

> **hexToBytes**(`hex`): `Uint8Array`

Decodes a non-empty even-length hexadecimal string into bytes.

## Parameters

### hex

`string`

Lowercase or uppercase hexadecimal input.

## Returns

`Uint8Array`

Decoded bytes.

## Throws

[InvalidPayloadError](../../core/classes/InvalidPayloadError/) When the input is not valid hexadecimal.
