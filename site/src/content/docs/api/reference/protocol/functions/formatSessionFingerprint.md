# Function: formatSessionFingerprint()

> **formatSessionFingerprint**(`transcriptHash`): `string`

Formats the first 32 hexadecimal characters of a transcript hash as a
session fingerprint for out-of-band comparison.

## Parameters

### transcriptHash

`string`

Lowercase hexadecimal transcript hash.

## Returns

`string`

Grouped session fingerprint string.
