Use the root package for all public imports. The package no longer ships supported subpath entrypoints.

## What lives here

- additive ElGamal encryption and decryption
- additive ciphertext combination
- key generation
- validation helpers
- threshold sharing and decryption helpers
- transport and authentication helpers
- protocol hashing, transcript, and verification helpers
- GJKR reducer and transcript-verification helpers

## Typical import style

```typescript
import {
    addEncryptedValues,
    decryptAdditive,
    deriveH,
    encryptAdditive,
    generateParameters,
} from "threshold-elgamal";
```

## Use the root package when

- you want the supported `ristretto255` workflow
- you want public additive, threshold, protocol, transport, or DKG helpers
- you want an import surface that will match the published package exports

## Explore the generated reference when

- you need exact signatures, types, and return shapes
- you want the full root export inventory
- you are wiring a larger voting or DKG workflow and need contract-level detail
