## What this library is for

`threshold-elgamal` is a browser-native TypeScript library for one shipped workflow:

- additive ElGamal on `ristretto255`
- honest-majority GJKR DKG
- fixed `1..10` score voting
- organizer-signed `ballot-close`
- full public-board verification

## What it does not try to do

This is a hardened research prototype, not audited production voting software. It does not claim coercion resistance, cast-as-intended guarantees against a compromised client, or constant-time JavaScript `bigint` arithmetic.
