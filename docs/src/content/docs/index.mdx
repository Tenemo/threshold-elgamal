## What this library is for

`threshold-elgamal` is a browser-native TypeScript library for finite-field ElGamal research prototypes. It ships additive ElGamal on the safe root package, dealer threshold sharing and decryption, VSS, proofs, typed protocol payloads, transport helpers, and log-driven DKG reducers.

## What it does not try to do

This is a hardened research prototype, not audited production voting software. It does not claim coercion resistance, cast-as-intended guarantees, or constant-time JavaScript `bigint` arithmetic.
