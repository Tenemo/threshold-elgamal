- Core finite-field helpers and RFC 7919 group definitions
- Deterministic `h` derivation and subgroup validation
- Web Crypto randomness with rejection sampling

- Schnorr, DLEQ, and disjunctive proofs
- Canonical manifest, payload, ordering, and transcript helpers
- Authenticated transport envelopes and dealer-challenge helpers
- Log-driven Joint-Feldman and GJKR reducers
